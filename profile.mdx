---
title: Profile
description: "Get Profile Details"
---

**URL:** `{{base_url}}/user_profile` <br />
**Method:** POST <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />

**Access_Token:** Replace this with the actual access token value

## Response Codes

| Code | Description                     |
| ---- | ------------------------------- |
| 200  | Successful                      |
| 404  | Error: Profile not found        |
| 400  | Server Error                    |
