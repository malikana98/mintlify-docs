---
title: Generate Bearer Token
description: 'Generate a Bearer using Developer Credentials.'
---

**URL:** `{{base_url}}/generate_token` <br />
**Method:** POST <br />

**Basic Auth:** Replace `UserID (Username)` with your actual Username and `API Key (Password)` with your actual Password.

## Headers
**Subscription:** Replace this with the actual subscription value

## Response Codes

| Code | Description          |
| ---- | -------------------- |
| 200  | Successful           |
| 400  | Error                |
| 401  | Denied Access        |
