## Authentication

All API endpoints are authenticated using a bearer token. 
A bearer token is generated using the endpoint: <br />
`https://dumacloud.com/apis/controller/generate_token`


1. User Account ID and API Key are used for Basic Authentication

2. The Subscription key is also put in the Headers

3. Generate the Request to Obtain the Bearer Token
