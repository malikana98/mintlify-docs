---
title: Refresh Token API
description: "Generate Access Token using Refresh Token"
---

**Method:** POST <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />

**Refresh_Token:** Replace this with the actual refresh token value

## Response Codes

| Code | Description                                            |
| ---- | ------------------------------------------------------ |
| 200  | Successful                                             |
| 400  | Error: Refresh token is invalid or has expired         |
| 401  | Refresh token expired. Sign in again                   |
