---
title: Return Sale
description: "Endpoint for making Sales Returns"
---

**URL:** `{{base_url}}/modules/revenue/reverse`  <br />

**Method:** POST <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />

**Access_Token:** Replace this with the actual access token value

## Request Body

| Key           | Values                          | Required |
| ------------- | ------------------------------- | -------- |
| uid           | External Sales Unique Id       | Yes      |
| code          | Code of the item you want to return        | Yes      |
| quantity      | Quantity you want to return        | Yes      |


## Response Codes

| Code | Description                        |
| ---- | ---------------------------------- |
| 200  | Successful                         |
| 500  | Internal Server Error              |
| 401  | Unauthorized: Check error message  |
