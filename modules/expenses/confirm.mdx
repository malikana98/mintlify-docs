---

**URL:** `{{base_url}}/modules/expenses/confirm` <br />

**Method:** POST <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />

**Access_Token:** Replace this with the actual access token value

## Request Body

| Key           | Values                          | Required |
| ------------- | ------------------------------- | -------- |
| expense_number | Enter the expense number        | Yes      |
| action        | Value should be `confirmed` or `declined` | Yes      |


## Response Codes

| Code | Description                        |
| ---- | ---------------------------------- |
| 200  | Successful                         |
| 500  | Internal Server Error              |
| 401  | Unauthorized: Check error message  |


Example JSON request body:
```json
{
    "expense_number": 1,
    "action": "confirmed"
}
