---
title: Expense Repayment
description: "Endpoint for making expenses repayment."
---

**URL:** `{{base_url}}/modules/expenses/repayment`  <br />

**Method:** POST <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />

**Access_Token:** Replace this with the actual access token value

## Request Body

| Key           | Values                          | Required |
| ------------- | ------------------------------- | -------- |
| amount   | Enter repayment amount   | Yes      |
| expense_number   | Enter expense number   | Yes      |
| payment_method   | Enter payment method. Value should either be `cash` or `bank`   | Yes      |

Below is an example of json request body

```json
    {
   "expense_number":2,
   "amount":2000,
   "payment_method":"bank"
  }


## Response Codes

| Code | Description                        |
| ---- | ---------------------------------- |
| 201  | Successful                         |
| 500  | Internal Server Error              |
| 401  | Unauthorized: Check error message  |
