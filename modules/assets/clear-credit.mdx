---
title: Clear Credit
description: "Endpoint for clearing credit status of assets bought on credit."
---

**URL:** `{{base_url}}/modules/assets/clear_credit`  <br />

**Method:** POST  <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value  <br />

**Access_Token:** Replace this with the actual access token value

## Request Body

| Key           | Values                          | Required |
| ------------- | ------------------------------- | -------- |
| asset_number  | Enter asset number              | Yes      |


## Response Codes

| Code | Description                        |
| ---- | ---------------------------------- |
| 200  | Successful                         |
| 500  | Internal Server Error              |
| 401  | Unauthorized: Check error message  |
