---
title: Get Equity
description: "Endpoint for getting equity using id."
---

**URL:** `{{base_url}}/modules/equity/getEquity`  <br />

**Method:** POST  <br />

**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value  <br />

**Access_Token:** Replace this with the actual access token value

## Request Body

| Key     | Values          | Required |
| ------- | --------------- | -------- |
| id      | Equity ID       | Yes      |


## Response Codes

| Code | Description                        |
| ---- | ---------------------------------- |
| 200  | Successful                         |
| 500  | Internal Server Error              |
| 401  | Unauthorized: Check error message  |

Example of request body

```json
{
    "id": 1
}
