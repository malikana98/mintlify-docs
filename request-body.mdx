---
title: Get Request Body
description: "Module APIs are dynamic. This endpoint lists the custom fields per account."
---

**URL:** `{{base_url}}/get_request_body` <br />
**Method:** POST <br />
**Bearer Token:** Replace this with an actual token in the request

## Headers
**Subscription:** Replace this with the subscription value <br />
**Access_Token:** Replace this with the actual access token value

## Request Body
| Key    | Values                            |
| ------ | --------------------------------- |
| module | Enter the name of the form module |

- `module`: String - The name of the form module.

## Response Codes

| Code | Description             |
| ---- | ----------------------- |
| 201  | Successful              |
| 404  | Error: Not found        |
