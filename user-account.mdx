---
title: Create Account
description: "API for creating a user account"
---

Method: POST
Bearer Token: <token>

##Headers
Subscription: <value>

## Request Body
 
|  Keys  | Values                          |        |
| -------- | ------------------------------------- |--------|
| forename     | Enter the forename of the user                     |  Required      |
| surname   | Enter the surname of the user                         |    Required    |
| email   | Email of the User |   Required     |    
| username   | The unique username for the user |    Required    |
| phoneNumber   | The phone number for the user |    Required    |
| businessName   | The name of the business |  Required      |  
| currency   | Enter the currency code(Int) |   Required     |  
| password   | Password with minimum of 8 characters |   Required     |
